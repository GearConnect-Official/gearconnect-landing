# Screenshots de l'application GearConnect

Ce dossier contient les screenshots de l'application à utiliser sur le site web.

## Screenshots requis

### Screenshots principaux (priorité haute)

1. **home-feed.png** - Écran d'accueil avec le feed social
   - Format: 9:19 (ratio téléphone)
   - Taille recommandée: 1080x2280px
   - Emplacement dans l'app: `app/(app)/(tabs)/home.tsx`

2. **feed.png** - Feed social avec publications
   - Format: 9:19
   - Taille recommandée: 1080x2280px
   - Emplacement dans l'app: `app/(app)/(tabs)/home.tsx` (HomeScreen)

3. **events.png** - Liste des événements
   - Format: 9:19
   - Taille recommandée: 1080x2280px
   - Emplacement dans l'app: `app/(app)/(tabs)/events.tsx` (EventsScreen)

4. **performances.png** - Écran de suivi des performances
   - Format: 9:19
   - Taille recommandée: 1080x2280px
   - Emplacement dans l'app: `app/(app)/performances.tsx`

### Screenshots secondaires (optionnels)

5. **profile.png** - Profil utilisateur
   - Emplacement dans l'app: `app/(app)/profile.tsx`

6. **publication-create.png** - Création de publication
   - Emplacement dans l'app: `app/(app)/(tabs)/publication.tsx`

7. **event-detail.png** - Détail d'un événement
   - Emplacement dans l'app: `app/(app)/eventDetail.tsx`

## Instructions pour capturer les screenshots

1. Utiliser un émulateur iOS ou Android
2. Prendre des captures d'écran en mode portrait
3. S'assurer que les données de démo sont cohérentes et professionnelles
4. Retirer les informations sensibles si nécessaire
5. Optimiser les images pour le web (compression)

## Utilisation dans le code

Les screenshots sont référencés dans `src/app/page.tsx` avec des placeholders.
Remplacer les placeholders par les vraies images une fois disponibles.
